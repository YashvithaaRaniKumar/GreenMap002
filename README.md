# GreenMap – Community Waste Awareness Platform

GreenMap is a beginner-friendly web application that helps identify local waste hotspots and promotes awareness about proper waste management practices in cities and towns.

This project is developed as part of the *1M1B Green Internship – Week 2 Capstone Project*.

---

## Problem Statement

Many communities face issues such as uncollected garbage, illegal dumping, and lack of awareness about waste segregation
